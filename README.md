# TRIVERGE

**Three colonies. One survives — or all three fall.**

An explorable 3D open world that runs right in your web browser — no install, no account. The entire world — terrain, ~190,000 trees, flow-carved rivers, volumetric clouds, real-time lighting — is generated by code the moment it loads. It was built almost entirely by **Claude Fable 5**, Anthropic's newest AI model.

## ▶ Download & play

### [⬇ Download TRIVERGE](https://github.com/mikrod2011-oss/triverge/releases/latest/download/TRIVERGE.zip)

1. Download the zip above and unzip it.
2. Double-click **`Triverge.html`**.
3. It opens in your browser and generates the world on your GPU (give it ~10 seconds — you'll see a loading bar).

> **Requires Google Chrome** (version 113 or newer). Safari and Firefox can't run it — the engine uses WebGPU, which today only Chrome/Chromium browsers support. If `Triverge.html` opens in another browser, right-click it → **Open With → Google Chrome**. (The zip also includes one-click launchers for Mac and Windows that force Chrome for you.)

### Controls
Click the page to capture your mouse · **WASD** move · **Shift** sprint · **Space** jump · **V** walk / fly · **E / Q** up / down (fly) · **mouse wheel** fly speed · **1–9** scenic viewpoints · **F** flythrough · **F3** stats

## The game

Three rival civilizations share one wild frontier, and each guards a self-destruct core buried inside its city. You defend your own core while racing to reach the other two. Trip an enemy's core and that civilization is out. **Last colony standing wins — but if no one wins before the 20-minute mark, every core goes off and all three fall together.**

This release is the **world foundation** — the procedural open world the match is fought over. Free-roam it today; the three-faction match systems build on top of it.
